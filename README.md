# Installation

> sudo apt-get update

> sudo apt-get install python-pip

> sudo pip install pip-update

> git clone https://github.com/Vaxure1337/LazySource

> cd LazySource

> pip install -r requirements.txt

> python setup.py install

> python LazySource.py
